## 学习主线

围绕「给大模型扩展能力、开发 Agent」展开，主要涉及：

- **Memory**：记忆管理
- **Tool**：工具调用（读写文件、执行命令、操作浏览器等）
- **RAG**：文档/知识库查询

技术栈：Node + LangChain（单 Agent）+ LangGraph（多 Agent），结合 Nest 后端做 AI 全栈。
